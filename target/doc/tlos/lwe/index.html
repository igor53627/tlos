Layer 2: LWE encryption for control function hiding.

Uses standard LWE with Gaussian noise (n=384, q=65521, σ=8) providing ~2^112 post-quantum security against lattice attacks.

Structs§

LweCiphertext
LWE ciphertext: a ∈ Z_q^n and b ∈ Z_q where b = ⟨a,s⟩ + e + m·(q/2).
